# Homework 6 - Implement Inheritance

## Overview

This project demonstrates inheritance in Java using a hierarchy of shape classes across two packages. A base `Shape` class is extended by `Rectangle` and `Circle`, which are further extended into `Square` and `SemiCircle`. The program creates objects to show how methods are inherited and used.

## What I Learned

* How to use the `extends` keyword
* How inheritance works across multiple levels
* How subclasses access parent class methods
