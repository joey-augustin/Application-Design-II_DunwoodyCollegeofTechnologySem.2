# Homework 4 - Smart Home

## Overview
The goal of this assignment was to build a console-based Java application that simulates a smart home system. Users can create and manage devices like Lights and Thermostats, which share common behaviors through an interface. The program stores devices in a collection and displays a summary of all entries.

## What I Learned
- Interfaces and polymorphism
- Using ArrayList to store objects
- Class design and constructors
- User input with Scanner

## Technologies
- Java
- ArrayList (Java Collections Framework)
- Scanner (User Input)
- Eclipse IDE
