# Homework 3 - Research and Development 2

## Overview
The goal of this assignment was to develop a console-based Java application that converts temperatures between Celsius and Fahrenheit. The program allows users to select a conversion type, input a temperature, and view the converted result. In addition to building the application, I implemented unit tests to verify the accuracy of the conversion methods.

## What I Learned
- Classes and Static Methods
- Package Organization and Structure
- User Input Handling with Scanner
- Unit Testing Fundamentals (Arrange, Act, Assert)
- Using JUnit for Automated Testing

## Technologies
- Java
- JUnit5
- Eclipse IDE
